Projects Adding a Medium feed to the site

I had started a blog on this site as Jekyll works well organising them on a static pages site, but found I wasn't really writing anything here. I've used Medium far more for writing actual blog posts, so wanted to bring that content in to this site rather than just link to it.

Medium publishes an RSS feed to syndicate content, but I needed a way to actually publish this on the site.

There were a few ways I found online to do it, but the first step was getting the RSS feed converted into json by using rss2json.com which is a free service.

After that I was a bit stuck as most of the advice I found was way too technical for me. Finally I found Giacinto Carlucci's excellent Medium on website repo providing just what I needed - a nice set of instructions and all the code set out for me

Giacinto's code provides the javascript along with css, both of which have options for how you want the blogroll to appear on your site. The js has four template options (classic, mini, compact and custom - the last being a 'do your own thing' approach) and the css has three themes (light, dark and 'batman'!)

The API provided by the rss2json site slots nicely into the API_URL constant in the js, and then you are good to go! I changed the template a bit to add my own button at the foot of each card, and used some of the site colours in the css file.

Issues

  • If I used an image from another source (like a youtube video) this displays an odd URL within the body of the card summary. I've not been able to resolve that one
