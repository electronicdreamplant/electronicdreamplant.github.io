Using GitHub Pages

I wanted to have a workspace I could play with while getting better at using Git, and found out (by accident really) that GitHub can also be used to host webpages via GitHub Pages. This felt like a perfect solution

To be started I followed this excellent guide by Johnathan McGlone to understand the basics. He's good on explaining the way Jekyll works, particularly for the blog part.

To assign my domain name I followed the GitHub help pages, which get you most of the distance.

More recently I've found this guide by RedWrite which has persuaded me to try out the GitHub Desktop to do things properly
