Projects/Live Analytics Dashboard

This project was kick-started by a LocalGovDigital Slack discussion, with Ben Cheetham trying to set up a display screen for his workplace.

Ben had come across a fork of a dashboard built for the Government Digital Service by Ed Sowden. The original project is here.

The nice part was this had been set up to run on Heroku so there's no mucking around with servers to get it running.

The fork came with some nice instructions but these proved too tricky for me to get my head around completely. Luckily my daughter had managed to land herself a handy webdev who was able to show me the way with some really good instructions.

I've run this on a large screen at work, which is particulary effective. The best occasion was during local elections, when we could see exactly how the event affected website traffic, and how much more traffic we were getting!

It's also really useful to check on what people are searching for at particular times of the day, or getting a sense of when mobile use overtakes desktop use.
